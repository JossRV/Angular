CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para dar estilo y formato a las páginas web.

Se utiliza para separar la presentación visual del contenido HTML y hacer que la estructura y el estilo de una página sean más fáciles de mantener. Esto se logra mediante la creación de reglas de estilo, que se aplican a los elementos HTML utilizando selectores y propiedades CSS.

Además de los selectores y las propiedades, CSS también utiliza valores y unidades para definir el estilo de los elementos HTML
